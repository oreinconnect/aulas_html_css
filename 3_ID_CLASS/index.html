ID e CLASS

ID

O ID funciona como um CPF. Assim como só posso ter um numero de cpf, cada elemento só pode ter um valor de ID. O CPF é unico, ou seja, ninguém pode um ter um CPF igual ao meu. Isso significa que o ID também é unico, então não é possivel repetir o mesmo valor duas vezes.

CLASS

O class funciona de maneira oposta ao ID. Diferente do ID em que você só pode ter um valor, no CLASS você pode ter vários valores. Também diferente do ID, em que não se pode repetir um valor, no CLASS você pode utilizar o mesmo valor para vários elementos.
